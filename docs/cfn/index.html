    Cloud Formation

    Cloud formation template defines Cloud Formation stack. Stack creates, updates, deletes resources based on template

    • Template an be written using JSON or YAML
    • Template an contain up to 200 resources
    • When Stack is deleted - all resources are deleted
    • Stack can be updated with new version of template
    • When logical resources added to templates new physical resources added to stack
    • When logical resources deleted - physical are deleted
    • When logical resources updated - physical can be updated or deleted (depends on resource)

    Template elements

    • Version
    • Description. Should go after version if defined
    • Metadata
    • Parameters
    • Mappings
    • Conditions
    • Transforms
